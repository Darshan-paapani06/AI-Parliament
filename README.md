## 🧬 THE ORIGIN STORY

Thousands of years ago, humans made a profound discovery:

**One mind is never enough.**

So we invented juries. Parliaments. Peer review. Scientific debate. Adversarial court systems. We built entire civilizations around the idea that **structured disagreement between diverse minds** produces better truth than any single authority.

Then came AI — and we forgot all of it.

We gave the world chatbots. Single models. One voice answering everything. One architecture, one training run, one perspective — confidently wrong in ways it can never detect about itself.

**Project Synapse is the correction.**

Five AI agents. Five completely different cognitive styles. Two rounds of structured debate. One synthesized verdict. This is not a chatbot. This is not a search engine. This is the first **AI Parliament** — and nothing like it exists anywhere on Earth.

<div align="center">

### The Problem With Every Other AI Tool

</div>

| Tool | What It Does | The Fatal Flaw |
|:---|:---|:---|

| Gemini | One model answers | Single architecture = single blind spot |
| Perplexity | One model + search | Still one reasoning engine |
| Claude | One model answers | One perspective, however good |
| **Synapse** | **5 agents debate** | **Adversarial cross-validation. No blind spots.** |

<div align="center">

### The Science Behind Why This Works

</div>

In 2015, Philip Tetlock published research on **Superforecasting** — the most rigorous study of human prediction ever conducted. The finding that shook the intelligence community:

> *Structured groups of diverse thinkers with conflicting viewpoints **dramatically and consistently** outpredict individual experts — including world-class specialists in their own domains.*

Project Synapse is this research, implemented in code.

The Devil's Advocate alone eliminates an entire class of AI failures — **sycophancy** and **overconfidence** — that plague every single-model system. When one agent is architecturally required to challenge every claim, the output is fundamentally more reliable.

**This is not a feature. This is a paradigm shift.**

## 🧩 THE 5 AGENTS — DEEP DIVE

<details>
<summary><b>🧠 THE REASONER — DeepSeek-R1 (671B)</b></summary>

**Cognitive Identity:** Pure deductive and inductive logic machine.

The Reasoner never accepts a claim at face value. Every argument gets decomposed into: *what are the premises? Are they true? Does the conclusion actually follow?*

Modeled after DeepSeek-R1's chain-of-thought architecture — the model that shocked the world in January 2025 by matching OpenAI o1 using reinforcement learning on reasoning traces.

**In the Parliament:** Establishes the logical skeleton of any debate. If an argument doesn't hold up under formal logical scrutiny, The Reasoner finds it.

🔗 [DeepSeek-R1 on GitHub](https://github.com/deepseek-ai/DeepSeek-R1) — ⭐ 89k stars

</details>

<details>
<summary><b>📚 THE SCHOLAR — Qwen 2.5 (72B)</b></summary>

**Cognitive Identity:** Cross-domain knowledge synthesizer.

The Scholar brings the weight of history, science, economics, and human experience to every debate. No argument exists in a vacuum — The Scholar finds the relevant precedents, analogies, and empirical patterns.

Modeled after Qwen 2.5, Alibaba's world-class open LLM trained on 18 trillion tokens, with exceptional multilingual and knowledge-retrieval capabilities.

**In the Parliament:** Grounds abstract arguments in concrete reality. "This is exactly what happened when..." is The Scholar's signature move.

🔗 [Qwen 2.5 on GitHub](https://github.com/QwenLM/Qwen2.5) — ⭐ 18k stars

</details>

<details>
<summary><b>⚗️ THE VALIDATOR — WizardCoder (34B)</b></summary>

**Cognitive Identity:** Empirical scientist demanding proof.

The Validator asks the question every scientist asks: *how would we test this? What evidence would change your mind? What is measurable vs. speculative?*

Modeled after WizardCoder — the model that defeated GPT-4 on HumanEval using Evol-Instruct methodology, known for precise, systematic, verifiable outputs.

**In the Parliament:** The immune system against unfalsifiable claims. If it can't be tested, The Validator calls it out.

🔗 [WizardCoder on GitHub](https://github.com/nlpxucan/WizardLM) — ⭐ 9k stars

</details>

<details>
<summary><b>👹 DEVIL'S ADVOCATE — Phi-3.5 (3.8B)</b></summary>

**Cognitive Identity:** Professional contrarian. Full-time flaw finder.

This is the most important agent in the Parliament. The Devil's Advocate is architecturally required to challenge every argument — even ones it privately agrees with. This eliminates the single biggest failure mode of AI: **sycophancy**.

Modeled after Microsoft's Phi-3.5 — the model that proved a tiny 3.8B parameter system can outthink models 10× its size when data quality is maximized. Sharp, fast, incisive.

**In the Parliament:** Asks the question nobody wants to hear. Finds the assumption everyone missed. Makes every other agent stronger by challenging them.

🔗 [Phi-3.5 on GitHub](https://github.com/microsoft/Phi-3CookBook) — ⭐ 7k stars

</details>

<details>
<summary><b>⚖️ THE SYNTHESIZER — Mistral 7B</b></summary>

**Cognitive Identity:** The final, impartial judge.

The Synthesizer has listened to everything. Both rounds. All 8 arguments. It now weighs them fairly — finding where agents agree (signal), where they conflict (important disagreement), and what the most defensible conclusion is.

Modeled after Mistral 7B — the model that redefined efficiency when it outperformed LLaMA 2 13B at half the size. Known for balanced, clear, decisive reasoning.

**In the Parliament:** Delivers the verdict. Always includes a confidence score (0–100%) calibrated to how much the debate agents agreed vs. clashed.

🔗 [Mistral 7B on GitHub](https://github.com/mistralai/mistral-src) — ⭐ 10k stars

</details>

## 🔧 CUSTOMIZATION GUIDE

### Add a New Agent

Open `src/agents/agents.js` and add to the `AGENTS` array:

```javascript
{
  id: "historian",
  name: "BLOOM 176B",
  role: "THE HISTORIAN",
  icon: "🏛️",
  color: "#bf5af2",
  glow: "rgba(191,90,242,0.4)",
  params: "176B",
  githubUrl: "https://github.com/bigscience-workshop/bloom",
  shortDesc: "Deep historical pattern recognition",
  persona: `You are THE HISTORIAN in the Synapse Parliament...
  [define their unique cognitive rules here]`,
}
```

### Plug In Real LLM APIs

Replace the Claude calls in `src/agents/api.js` with real model endpoints:

```javascript
// DeepSeek API (dirt cheap — $0.001/1K tokens)
const response = await fetch("https://api.deepseek.com/v1/chat/completions", {
  headers: { "Authorization": `Bearer ${DEEPSEEK_KEY}` },
  body: JSON.stringify({ model: "deepseek-reasoner", messages: [...] })
});

// Or run 100% locally with Ollama (free forever)
const response = await fetch("http://localhost:11434/api/generate", {
  body: JSON.stringify({ model: "mistral", prompt: yourPrompt })
});
```

### Run With Local Models (Zero API Cost)

```bash
# Install Ollama — https://ollama.com
ollama pull mistral        # 4GB  — The Synthesizer
ollama pull phi3           # 2GB  — Devil's Advocate  
ollama pull qwen2.5:7b     # 4GB  — The Scholar
ollama pull wizardcoder    # 4GB  — The Validator
```
